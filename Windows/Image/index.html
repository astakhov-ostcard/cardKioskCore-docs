Image¶

Отображает изображение из файловой системы. Поддерживает анимацию вращения и локализацию.

{
  "type": "Image",
  "name": "img_logo",
  "margin": { "left": 300, "top": 100 },
  "size": { "width": 480, "height": 200 },
  "customProperties": {
    "source": "logo.png",
    "stretch": "Uniform"
  }
}

CustomProperties¶

Ключ Тип Описание
source string Путь к файлу изображения. Относительный путь разрешается от папки Images/ внутри ресурсов. Поддерживает абсолютный путь
stretch string Режим масштабирования: "Uniform" (по умолчанию), "Fill", "UniformToFill", "None"
rotate double Скорость вращения в градусах в секунду. Положительное — по часовой, отрицательное — против. Например: "90" = один оборот за 4 секунды

Дополнительные параметры¶

Параметр Тип Описание
localized bool Если true — путь к изображению строится как Images/{язык}/{source}. При смене языка изображение обновляется автоматически
tint string Цветовой фильтр поверх изображения (HEX). Применяет эффект тонирования

Если файл не найден или повреждён — вместо него отображается изображение-заглушка из ресурсов приложения.
