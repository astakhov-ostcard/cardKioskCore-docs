# Локализация — Инструменты редактора

## Автодополнение ключей переводов

При вводе значения для поля `translationKey` в JSON-редакторе автоматически появляется список доступных ключей из активного словаря переводов:

```json
"translationKey": "▌"  ← здесь откроется autocomplete
```

Список формируется из реально загруженного файла `Translations.xlsx`. Это позволяет видеть только существующие ключи и не допускать опечаток.

---

## Получить все ключи переводов из JSON

**Меню:** `Tools → Get all translation keys`

Сканирует весь открытый JSON-файл и собирает все значения полей `translationKey`, а также ключи из команд `error:` и `warning:`. Результат копируется в буфер обмена в виде отсортированного списка (один ключ на строку).

Используется для быстрой передачи списка ключей переводчику или для сверки с таблицей.

**Что извлекается:**
- Все поля с именем `translationKey` во всём дереве JSON
- Ключи из строк вида `"error:key.title|key.desc"` и `"warning:key.title|key.desc"`

---

## Найти отсутствующие ключи

**Меню:** `Tools → Get missing translation keys`

Выполняет ту же сборку ключей из JSON, затем сравнивает их с текущим загруженным словарём. Ключи, которых нет в `Translations.xlsx`, копируются в буфер обмена.

Позволяет быстро выявить пропущенные переводы перед релизом:

1. Открыть нужный JSON в редакторе
2. Запустить команду
3. Вставить результат из буфера — это список ключей, которые нужно добавить в таблицу

---

## Импорт переводов из Google Sheets

**Меню:** `Tools → Import translations from Google Sheets`

Позволяет обновить `Translations.xlsx` прямо из редактора, не открывая Excel вручную.

**Порядок действий:**

1. Открыть Google Таблицу с переводами
2. Дать доступ «Просматривать» для всех, у кого есть ссылка
3. В редакторе: `Tools → Import translations from Google Sheets`
4. Вставить URL таблицы в диалог
5. Редактор скачает и сохранит в `<ResourcesFolder>\Translations.xlsx`

**Требования к таблице:**
- Первая строка — заголовки: первая ячейка любая, далее ключи языков (`ru`, `uz`, `en`, ...)
- Первый столбец — ключи переводов
- Остальные ячейки — тексты

## Структура команды `language:` — краткая справка

| Команда | Описание |
|---|---|
| `language:ru` | Переключить на язык `ru` |
| `language:session:varName` | Язык берётся из `SessionData["varName"]` |
| `language:persistent:varName` | Язык берётся из `PersistentData["varName"]` |

Смена языка сохраняется в `PersistentData["-language-"]` автоматически.
