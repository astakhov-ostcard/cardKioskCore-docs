## Border

Прямоугольник — используется как декоративный фон, разделитель или контейнер.

```json
{
  "type": "Border",
  "name": "bg_card",
  "margin": { "left": 40, "top": 300, "right": 40 },
  "size": { "height": 400 },
  "border": {
    "background": "#1E293B",
    "borderBrush": "#334155",
    "borderThickness": 2,
    "cornerRadius": 24
  }
}
```

### Параметры

| Параметр | Тип | Описание |
|---|---|---|
| `border` | BorderOptions | Обязательный для визуального отображения. Задаёт фон, рамку, скругление и выравнивание |

> Border не поддерживает текст, клики и дочерние элементы — это чисто визуальный слой.
