## QR

Генерирует QR-код из текстового значения. Поддерживает динамическое обновление через сессию.

```json
{
  "type": "Qr",
  "name": "qr_payment",
  "margin": { "left": 340, "top": 600 },
  "size": { "width": 400, "height": 400 },
  "customProperties": {
    "value": "session:paymentUrl",
    "foreground": "#000000"
  }
}
```

### CustomProperties

| Ключ | Тип | По умолчанию | Описание |
|---|---|---|---|
| `value` | string | `"undefined"` | Текст, из которого генерируется QR-код. Поддерживает `session:varName` — QR обновляется автоматически при изменении переменной. Также поддерживает ключи локализации |
| `foreground` | string | `"#000000"` | Цвет модулей QR-кода в формате HEX. Фон всегда прозрачный |
