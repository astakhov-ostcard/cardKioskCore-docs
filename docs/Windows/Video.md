## Video
Воспроизводит видеофайл в бесконечном цикле. Поддерживает отложенную загрузку.

```json
{
  "type": "Video",
  "name": "introClip",
  "position": { "x": 0, "y": 0 },
  "size": { "width": 1080, "height": 1920 },
  "customProperties": {
    "source": "Videos/screensaver.mp4",
    "lazyLoad": "false"
  }
}
```

### CustomProperties
| Ключ | Тип | По умолчанию | Описание |
|---|---|---|---|
| `source` | string | — | Путь к видеофайлу. Относительный путь разрешается от папки `Videos/` внутри ресурсов. Поддерживает абсолютный путь |
| `lazyLoad` | string (`"true"` / `"false"`) | `"false"` | Если `"true"` — видео загружается и воспроизводится только когда элемент становится видимым. При скрытии элемента видео выгружается. Если `"false"` — видео загружается сразу при создании экрана |

**Особенности:**
- Видео воспроизводится автоматически в бесконечном цикле без звука (Volume = 0)
- Растяжение — `UniformToFill` (заполняет всю область с обрезкой)
- Для фоновых видео рекомендуется `lazyLoad: "false"`, чтобы избежать задержки воспроизведения
